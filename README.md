# Abinesh’s Plant House – React + Redux

A simple React + Redux shopping cart application for plants. Built with Vite and Redux Toolkit.

## Scripts
```bash
npm install
npm run dev    # start locally
npm run build  # production build
npm run preview
```

## Features
- Landing page with background and CTA
- Product list with 6 plants in 3 categories
- Add to Cart (disables when added)
- Header with cart count
- Cart page: increment, decrement, delete, totals
- Continue Shopping & Checkout (Coming Soon)
